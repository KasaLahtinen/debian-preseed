# Debian Preseed Form Designer

This is a modern React application powered by Vite. It serves as a dynamic User Interface for generating `preseed.cfg` files.

## Features
- Paste the JSON schema generated by the `parser.py` tool.
- Automatically generates interactive HTML forms with dropdowns, multiselects, and text inputs based on the Debian Installer schema.
- Outputs a valid `preseed.cfg` snippet that you can inject into your ISO.

## Getting Started

To run the development server:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```

## Architecture
The application consists of two main components:
- `PreseedLandingPage.jsx`: The main workspace where you paste the parser JSON and view the generated configuration.
- `PreseedForm.jsx`: The dynamic engine that recursively renders form fields based on the input schema.
